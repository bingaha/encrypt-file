# Vault - 文件加密工具

Windows 桌面文件/文件夹加密工具。编译为单个 exe，无需安装。

## 功能

### 目录模式

将 exe 放入目标目录运行，自动识别加密状态：

- **加密：** 设置密码后，递归加密目录下所有文件和子目录名，可选隐藏文件夹
- **解密：** 输入密码后，还原所有文件内容、文件名、目录名和修改时间

加密后可以带走 exe，目录只剩乱码文件名和不可读的文件内容。再次把 exe 放回来运行即可解密。

### 单文件模式

选择单个文件进行加密/解密，加密后文件自包含所有还原信息：

- 可以单独发送加密文件
- 接收方只需相同的 exe 和密码即可解密还原（包括原始文件名）

### 加密方案

- **大文件（> 128KB）：** 只覆写文件头部和尾部各 64KB，中间内容不动，速度接近瞬间
- **小文件（<= 128KB）：** 加密整个文件内容
- **文件名/目录名：** AES 加密后 hex 编码，不泄露原始名称
- **元数据：** 追加在文件末尾（约 100~300 字节），包含原始文件名、大小、时间戳
- **密钥派生：** PBKDF2-SHA256（100,000 次迭代）+ 每文件独立随机 salt
- **加密算法：** AES-256-CTR

### 文件夹隐藏

可选功能，加密后通过 Windows CLSID 重命名技术隐藏文件夹，在资源管理器中不可见（第三方工具如 Everything 仍可见，但文件内容已加密不可读）。

### 加密前校验

- 文件是否被占用/锁定
- 加密后文件名是否超出系统限制
- 是否已加密（防止重复加密）

## 构建

```bash
cargo build --release
```

产出：`target/release/vault.exe`（约 4MB）

## 使用方式

### 目录加密

1. 将 `vault.exe` 复制到要加密的目录
2. 双击运行
3. 程序自动检测到未加密状态，显示加密界面
4. 输入密码并确认，可选勾选"加密后隐藏文件夹"
5. 点击"加密"
6. 加密完成后可以把 exe 带走

### 目录解密

1. 将 `vault.exe` 放回加密目录
2. 双击运行
3. 程序自动检测到已加密状态，显示解密界面
4. 输入密码，点击"解密"
5. 所有文件和目录恢复原状

### 单文件加密

1. 运行 `vault.exe`
2. 切换到"单文件模式"
3. 选择文件，设置密码，点击"加密"

### 单文件解密

1. 运行 `vault.exe`
2. 切换到"单文件模式"
3. 选择加密文件，输入密码，点击"解密"

## 技术栈

- **语言：** Rust
- **GUI：** egui / eframe
- **加密：** AES-256-CTR + PBKDF2-SHA256
- **目标平台：** Windows
